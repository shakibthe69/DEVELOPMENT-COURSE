# JavaScript Chapter 19 - Events

## 📚 Overview

This project demonstrates how JavaScript Events work by building an interactive **Student Dashboard**.

Instead of learning each event separately, this project combines the most commonly used DOM events into one real-world application.

# 📌 Events Covered

| Event | Description |
|--------|-------------|
| click | Button click |
| dblclick | Double click |
| mouseover | Mouse enters element |
| mouseout | Mouse leaves element |
| mousedown | Mouse button pressed |
| mouseup | Mouse button released |
| mousemove | Mouse movement |
| keydown | Keyboard key pressed |
| keyup | Keyboard key released |
| input | User typing |
| change | Select value changed |
| focus | Input focused |
| blur | Input loses focus |
| submit | Form submission |
| preventDefault() | Prevent page refresh |

## Event Object

```javascript
button.addEventListener("click", (event) => {
    console.log(event.target);
});
```

Useful properties:

- event.target
- event.type
- event.key
- event.code
- event.clientX
- event.clientY
