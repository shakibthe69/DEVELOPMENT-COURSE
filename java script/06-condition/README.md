## if

```javascript
if(age >= 18){
    console.log("Adult");
}
```

---

## if...else

```javascript
if(age >= 18){
    console.log("Adult");
}else{
    console.log("Minor");
}
```

---

## switch

```javascript
switch(day){
    case 1:
        console.log("Monday");
        break;
}
```
