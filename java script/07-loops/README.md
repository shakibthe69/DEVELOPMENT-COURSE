## for Loop

```javascript
for(let i = 1; i <= 5; i++){
    console.log(i);
}
```

---

## while Loop

```javascript
let i = 1;

while(i <= 5){
    console.log(i);
    i++;
}
```

---

## do...while Loop

```javascript
let i = 1;

do{
    console.log(i);
    i++;
}while(i <= 5);
```
