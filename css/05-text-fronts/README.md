# Chapter 05 - Text & Fonts

## Objective

Learn how to style text professionally using CSS.

---

# Font Family

```css
font-family: Arial, sans-serif;
```

Changes the font style.

---

# Font Size

```css
font-size:20px;
```

Controls the text size.

---

# Font Weight

```css
font-weight:bold;
```

Common values:

- 300
- 400
- 500
- 600
- 700

---

# Font Style

```css
font-style:italic;
```

---

# Text Alignment

```css
text-align:center;
```

Options:

- left
- center
- right
- justify

---

# Text Decoration

```css
text-decoration:none;
```

Examples:

- underline
- overline
- line-through

---

# Text Transform

```css
text-transform:uppercase;
```

Options:

- uppercase
- lowercase
- capitalize

---

# Letter Spacing

```css
letter-spacing:2px;
```

---

# Word Spacing

```css
word-spacing:10px;
```

---

# Line Height

```css
line-height:1.8;
```

---

# Text Shadow

```css
text-shadow:2px 2px 5px gray;
```

---

# Google Fonts

Example:

```html
<link href="https://fonts.googleapis.com/..." rel="stylesheet">
```

---

# Text Overflow

```css
text-overflow:ellipsis;
```

---

# Practice

- Create a blog article.
- Style a heading.
- Use Google Fonts.
- Add text shadow.
- Experiment with spacing.
