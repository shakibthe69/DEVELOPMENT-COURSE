# 02 - CSS Syntax

## Objective

Learn the basic syntax of CSS.

---

## CSS Rule

```css
selector {

    property: value;

}
```

Example:

```css
h1 {

    color: blue;

}
```

---

## Important Terms

### Selector

Selects an HTML element.

Example:

```css
p {

}
```

---

### Property

Describes what you want to change.

Examples:

- color
- font-size
- background-color
- padding

---

### Value

Specifies the setting for a property.

Examples:

```css
color: red;

font-size: 20px;
```

---

## Comments

```css
/* This is a comment */
```

---

## Best Practices

- Use proper indentation.
- Add comments for sections.
- One property per line.
- End every declaration with `;`.

---

## Practice

- Style a heading.
- Style a paragraph.
- Style a button.
- Experiment with different colors.
