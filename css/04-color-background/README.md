# 04 - Colors & Background

## Objective

Learn different color formats and background properties in CSS.

---

# Color Formats

## Color Name

```css
color: red;
```

---

## HEX

```css
color: #3498db;
```

---

## RGB

```css
color: rgb(255,0,0);
```

---

## RGBA

```css
color: rgba(255,0,0,0.5);
```

RGBA allows transparency.

---

## HSL

```css
color: hsl(120,100%,50%);
```

---

# Background Properties

```css
background-color
background-image
background-size
background-position
background-repeat
```

---

# Gradients

## Linear

```css
background: linear-gradient(to right, blue, purple);
```

## Radial

```css
background: radial-gradient(circle, yellow, orange);
```
