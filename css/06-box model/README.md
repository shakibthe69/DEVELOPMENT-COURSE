# Chapter 06 - CSS Box Model

## Objective

Learn how spacing works in CSS.

---

## Box Model

Every element consists of:

- Content
- Padding
- Border
- Margin

---

## Width

```css
width:300px;
```

---

## Height

```css
height:200px;
```

---

## Padding

```css
padding:20px;
```

Inside spacing.

---

## Border

```css
border:2px solid black;
```

---

## Margin

```css
margin:20px;
```

Outside spacing.

---

## Border Radius

```css
border-radius:10px;
```

---

## Box Sizing

```css
box-sizing:border-box;
```

Recommended for all projects.
