Welcome to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create web pages.

What You Will Learn

  • HTML document structure

Example

Hello World!

HTML Structure

<!DOCTYPE html>
<html>
    <head>
        <title>My Website</title>
    </head>

    <body>
        Hello World
    </body>
</html>

Explanation

Every HTML document starts with <!DOCTYPE html>.

The <html> tag is the root of the webpage.

The <head> contains information about the page.

The <body> contains everything visible on the webpage.

Browser Output

The browser ignores most spaces and indentation.

It only displays the content inside the body tag.
