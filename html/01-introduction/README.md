# Chapter 01 - Introduction to HTML

## What is HTML?

HTML stands for **HyperText Markup Language**.

It is the standard language used to build webpages.

HTML is **not** a programming language.

It is a markup language.

---

# Why Learn HTML?

Every website starts with HTML.

Examples:

- Google
- Facebook
- YouTube
- Amazon

All of these websites use HTML.

---

# How Does HTML Work?

You write HTML code.

↓

Browser reads the code.

↓

Browser displays the webpage.

---

# Basic HTML Structure

```html
<!DOCTYPE html>
<html>

<head>

<title>My Website</title>

</head>

<body>

Hello World

</body>

</html>
```

---

# Meaning of Each Tag

## <!DOCTYPE html>

Tells the browser this is an HTML5 document.

---

## <html>

Root element.

Everything is inside this tag.

---

## <head>

Contains:

- Title
- Meta tags
- CSS
- JavaScript links

Not visible on the webpage.

---

## <body>

Contains everything visible.

Example:

- Heading
- Paragraph
- Images
- Buttons
- Tables

---

# HTML Tags

Examples:

```html
<h1></h1>

<p></p>

<img>

<a></a>

<div></div>
```

---

# HTML Elements

Example:

```html
<p>Hello World</p>
```

Element = Opening Tag + Content + Closing Tag

---

# HTML Attributes

Example

```html
<img src="cat.jpg" alt="Cat">
```

Attributes give extra information.

Examples:

- src
- href
- alt
- id
- class
