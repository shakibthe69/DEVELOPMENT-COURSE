# Chapter 03 - HTML Text Formatting

## Learning Objectives

Learn how to format text using HTML elements.

---

# Bold Text

```html
<b>Bold Text</b>

<strong>Important Text</strong>
```

- `<b>` makes text bold.
- `<strong>` indicates strong importance (semantic meaning).

---

# Italic Text

```html
<i>Italic</i>

<em>Emphasized</em>
```

- `<i>` changes the appearance.
- `<em>` adds emphasis and is more meaningful for accessibility.

---

# Underline

```html
<u>Underline</u>
```

---

# Highlight

```html
<mark>Highlighted Text</mark>
```

---

# Small Text

```html
<small>Small Text</small>
```

---

# Deleted Text

```html
<del>Old Price</del>
```

Example:

```html
<del>$100</del> $80
```

---

# Inserted Text

```html
<ins>New Text</ins>
```

---

# Subscript

```html
H<sub>2</sub>O
```

Output:

H₂O

---

# Superscript

```html
x<sup>2</sup>
```

Output:

x²

---

# Code

```html
<code>
console.log("Hello");
</code>
```

Used for short code snippets.

---

# Preformatted Text

```html
<pre>
Line One
Line Two
</pre>
```

Preserves spaces and line breaks.

---

# Keyboard Input

```html
<kbd>Ctrl</kbd> + <kbd>S</kbd>
```

---

# Variable

```html
<var>x</var>
```

---

# Sample Output

```html
<samp>Hello World</samp>
```

---

# Abbreviation

```html
<abbr title="HyperText Markup Language">
HTML
</abbr>
```

Hover over the abbreviation to see its full meaning.

---

# Address

```html
<address>
Dhaka, Bangladesh
</address>
```

Typically used for contact information.

---

# Quotations

Inline quote:

```html
<q>Hello</q>
```

Block quote:

```html
<blockquote>
Practice makes perfect.
</blockquote>
```

---

# Best Practices

✅ Prefer `<strong>` over `<b>` when the text is important.

✅ Prefer `<em>` over `<i>` when the text needs emphasis.

✅ Use `<code>` for code snippets.

✅ Use `<mark>` only when highlighting important content.
